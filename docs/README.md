# MeetMind

MeetMind is an AI-powered meeting intelligence platform that enables users to securely upload meeting recordings, manage meetings, and automatically generate transcripts, summaries, and action items using AI.

## Features

- JWT Authentication
- Secure Meeting Management
- Audio Upload
- AI Transcription
- AI-generated Summaries
- Key Decision Extraction
- Action Item Extraction
- Meeting Dashboard

## Tech Stack

- React
- Node.js
- Express
- TypeScript
- MongoDB
